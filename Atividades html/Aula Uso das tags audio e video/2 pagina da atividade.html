Tags Audio e Video no HTML

O que é a tag <audio>?

A tag <audio> permite adicionar sons ou músicas diretamente em uma página HTML. O usuário pode ouvir o conteúdo sem precisar baixar o arquivo.

Exemplo:

Principais atributos da tag Audio

Atributo Função
controls Mostra os controles do player.
autoplay Inicia o áudio automaticamente.
loop Repete o áudio quando termina.
muted Começa o áudio sem som.
preload Define como o navegador carrega o áudio.

O que é a tag <video>?

A tag <video> permite inserir vídeos diretamente na página HTML, sem depender do YouTube.

Exemplo:

Principais atributos da tag Video

Atributo Função
controls Exibe os controles do vídeo.
autoplay Inicia o vídeo automaticamente.
loop Repete o vídeo.
muted Inicia o vídeo sem áudio.
poster Mostra uma imagem antes do vídeo iniciar.
width Define a largura do vídeo.
height Define a altura do vídeo.
